Documentation

Getting Started

  1. Install KSAT from the Downloads page.
  2. Open Settings → click Get a key to obtain your Steam Web API key. Paste it and press Save.
  3. Click Connect, sign in with Steam, and your library will appear.

Core Features

  • Notes & Tags: per-game notes; tags like cleanup, DLC1.
  • Filters: unlocked/locked/secret/manual-only + text search.
  • Sorting: default, next easiest, rarest first.
  • Friend Compare: paste SteamID64 or vanity; see differences.
  • Manual Overrides: mark achievements manually when APIs lag.
  • Launch/Store: launch a game or open its store page.
  • Share Card: export a clean progress card as PNG.
  • Themes: Neon, Midnight, Cozy, Solar + extra fun themes.

Troubleshooting

Blank Steam sign-in window

Install the Microsoft WebView2 Runtime and retry. Ensure your firewall allows KSAT.

Antivirus false positive

Can happen with unsigned installers. Download only from GitHub Releases; verify the hash.

Dev: “Waiting for http://localhost:5173/...”

Make sure Tauri starts Vite via beforeDevCommand or run Vite in another terminal.
